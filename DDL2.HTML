CREATE COMMAND

CREATE : The create table command is used to create tables to store the data.Tables are owned by the user who creates them.The column names in the table must be unique.

SYNTAX: 

CREATE TABLE table_name
                (
                    column_name1 datatype,
                    column_name2 datatype,
                    ...
                    column_nameN datatype
                );

EXAMPLE:

To create a table STUDENT which has following columns:

COLUMN NAME DATATYPE DESCRIPTION
Name varchar Name of the student
Roll no. int Roll no. of the student 
    CREATE TABLE STUDENT(
     NAME varchar(30),
     Roll_no int
    );
