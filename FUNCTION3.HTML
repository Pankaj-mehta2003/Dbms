SUM()

In SQL, the SUM() function is used to calculate the sum of values in a column. It is often used in conjunction with the SELECT statement to retrieve aggregated data from a table.



EXAMPLE

Consider below "Student" table for understanding the functioning of the SUM() function.
Name Roll_nO MARKS
MONA 1 100
SONA 2 90
NAINA 3 80
KAIRA 4 90
If you want total marks of all the students in the student table. 
    SELECT SUM(MARKS) AS TOTAL_MARKS FROM STUDENT;
    OUTPUT:
    

        
TOTAL_MARKS
