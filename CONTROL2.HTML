CONDITIONAL CONTROL

The structure of the PL/SQL IF statement is similar to the structure of the IF statements in other procedural languages. There are three forms of IF statements:


1.IF-THEN-END-IF

SYNTAX: 

    IF condition THEN
          statements;
    END IF;

EXAMPLE 


    DECLARE
    X NUMBER(2);
    BEGIN
    X:=5;
    IF X>0 THEN
    DBMS_OUTPUT.PUT_LINE('POSITIVE NUMBER');
    END IF;
    END;
                


    OUTPUT:
    POSITIVE NUMBER

2.IF-THEN-ELSE-END IF

SYNTAX: 

    IF condition THEN
              statements;
    ELSE 
              statements;
    END IF;

EXAMPLE 


    DECLARE
    X NUMBER(2);
    BEGIN
    X:=-25;
    IF X>0 THEN
    DBMS_OUTPUT.PUT_LINE('POSITIVE NUMBER');
    ELSE
    DBMS_OUTPUT.PUT_LINE('NEGATIVE NUMBER');   
    END IF;
    END;
                    


    OUTPUT:
    NEGATIVE NUMBER

3..IF-THEN--ELSIF-END IF

SYNTAX: 

        IF condition THEN
                  statements;
        ELSIF condition THEN 
                  statements;
        [ELSE 
                  statements;]
        END IF;

EXAMPLE 


    DECLARE
    X NUMBER(2);
    BEGIN
    X:=0;
    IF X>0 THEN
    DBMS_OUTPUT.PUT_LINE('POSITIVE NUMBER');
    ELSIF X=0 THEN
    DBMS_OUTPUT.PUT_LINE('ZERO'); 
    ELSE
    DBMS_OUTPUT.PUT_LINE('NEGATIVE NUMBER');     
    END IF;
    END;
                        


    OUTPUT:
    ZERO
