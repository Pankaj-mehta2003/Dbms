MINUS

The MINUS operator returns the rows unique to first query. It returns the remaining rows when the rows retrieved by the second query are subtracted from the rows retrieved by the first query.

SYNTAX

SELECT Column_list from Table1
minus
SELECT Column_list from Table2
[ORDER BY CLAUSE];
Consider the following table 'EMP':
DEPTNO ENAME JOB
10 CLARK ANALYST
20 SMITH CLERK
20 JONES MANAGER
30 ALLEN SALESMAN
30 JAMES CLERK
To display the jobs unique to department 20 'EMP' table: 
SELECT JOB FROM EMP
WHERE DEPTNO = 20
MINUS
SELECT JOB FROM EMP
WHERE DEPTNO IN(10,30);
OUTPUT:

JOB
MANAGER
