UPDATE COMMAND

The UPDATE statement is used to update oor modify the exiting rows in a table. Updation can be carried out for all the rows in a table or selected rows. The update statement modifies specific rows if the WHERE clause is specified. If you omit the WHERE clause, then all the rows in the table are modified.

SYNTAX: 

    UPDATE table_name SET column_name=val1,column_name2=val2.....
    [where condition];

EXAMPLE

Consider a table 'EMPLOYEE' with following values:

EMP_ID NAME SALARY DEPT
1 MONA 200000 HR
2 SONA 100000 SALES
3 NAINA 50000 SALES
4 NAMAN 150000 PRODUCTION


TO UPDATE SALARY OF EMPLOYEES TO 150000 OF EMPLOYEE WHOSE DEPARTMENT IS 'SALES': 
    UPDATE EMPLOYEE SET SALARY=150000 WHERE dept='SALES';
select * from EMPLOYEE;
