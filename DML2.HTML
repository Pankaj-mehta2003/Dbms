INSERTING VALUES INTO A TABLE

The INSERTstatement is used to insert new rows or records into a table.Values can be inserted for all the columns or for the selected columns of the table. You can insert data into a table through different ways:

1.INSERTING THE DATA WITHOUT SPECIFYING COLUMN NAME

SYNTAX: 

    INSERT INTO table_name values(val1,val2,-----valN);

EXAMPLE

Consider a table STUDENT which has following columns:

COLUMN NAME DATATYPE DESCRIPTION
Name varchar Name of the student
Roll no. int Roll no. of the student
The following SQL statement inserts value into table "STUDENT": 
    INSERT INTO STU VALUES('MONA',1);
Now to view the content of table execute following command: 
select * from STUDENT;
Name Roll_no
MONA 1

2.INSERTING THE DATA BY SPECIFYING COLUMN NAMES

SYNTAX: 

    INSERT INTO table_name [col1,col2----colN] values(val1,val2,-----valN);

EXAMPLE

The following SQL statement inserts value into table "STUDENT": 
    INSERT INTO STU (Name) VALUES('SONA');        
select * from STUDENT;
Name Roll_nO
MONA 1
SONA NULL
