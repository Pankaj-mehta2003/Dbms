Function execution

A function may accept one or many parameters, but must return a single value. You invoke functions as part of PL/SQL expressions, using variables to hold the returned value.

EXAMPLE 

    CREATE OR REPLACE FUNCTION F1(X NUMBER, Y NUMBER) 
    RETURN NUMBER IS
    NUM NUMBER;
    BEGIN
    NUM:=X+Y;
    RETURN (NUM);
    END F1;
    To call the above mentioned function:
    DECLARE 
    NUM1 NUMBER:=10;
    NUM2 NUMBER:=40;
    BEGIN 
    DBMS_OUTPUT.PUT_LINE(F1(NUM1,NUM2));
    END;
    

    OUTPUT:50
