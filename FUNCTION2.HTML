COUNT()

This function returns the number of rows in the table that satisfies the condition specified in the WHERE clause. If the WHERE clause is not specified, then the query returns the total number of rows in the table.



EXAMPLE

Consider below "Student" table for understanding the functioning of the Count() function.
Name Roll_nO MARKS
MONA 1 100
SONA 2 90
NAINA 3 80
KAIRA 4 90
If you want to count total no. of students in the student table. 
    select count(*) AS TOTAL_STUDENTS FROM STUDENT;
    OUTPUT:
    

        
TOTAL_STUDENTS
