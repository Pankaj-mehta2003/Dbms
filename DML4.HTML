DELETE COMMAND

The DELETE statement is used to delete rows from a table. To delete rows from a table, the table must be in your own schema or you must have the delete privilige on the table.


SYNTAX: 

    DELETE FROM table_name;
    [WHERE CONDITION];

Specified rows are deleted if you specify the WHERE clause in the DELETE statement. If you omit the WHERE clause, then all rows in the table are deleted.

EXAMPLE

1. Delete all records from table 'EMPLOYEE': 
    DELETE from EMPLOYEE;
2. Delete all records of sales department from EMPLOYEE table; 
    DELETE from EMPLOYEE where dept='SALES';
