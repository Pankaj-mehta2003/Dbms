GRANT COMMAND

GRANT command is used to provide system priviliges as well as object priviliges to user.


SYNTAX: 

    GRANT [type_of_priviliges] ON [user_name].[table name] TO [user_name]
    [WITH GRANT OPTION];
    In the syntax:
    WITH GRANT OPTION - allows a user to grant access rights to other users.

EXAMPLE

This command grants the SELECT, INSERT, and DELETE permissions on the database 'mydb' to 'user1'. 
    GRANT SELECT, INSERT, DELETE ON mydb TO 'user1'@'localhost';
