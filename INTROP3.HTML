ALTER COMMAND

ALTER : The alter table command is used to modify the definition(structure) of the table by modifying the definition of its columns.

1. ADDING TABLE COLUMNS

SYNTAX: 

ALTER TABLE table_name ADD column_name datatype;

EXAMPLE:

Consider a table STUDENT with following columns:

COLUMN NAME DATATYPE DESCRIPTION
Name varchar Name of the student
Roll no. int Roll no. of the student
The following SQL adds the "class" column in the "student" table: 
ALTER table STUDENT ADD class varchar(5);


2. Dropping table columns

SYNTAX: 

ALTER TABLE table_name DROP column_name;

EXAMPLE:

Now in the previous table we will delete the column class.
The following SQL drops the "class" column in the "student" table: 
ALTER table STUDENT DROP class;


3. Modifying table columns

SYNTAX: 

ALTER TABLE table_name MODIFY column_name datatype;

EXAMPLE:

Now in the previous table we will change the datatype of column roll_no.
The following SQL will change datatype of the "roll_no" column in the "student" table: 
ALTER table STUDENT_info MODIFY roll_no tinyint;
