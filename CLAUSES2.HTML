ORDER BY CLAUSE

The ORDER BY clause is used to sort the result-set in ascending or descending order.

SYNTAX: 

    SELECT column1, column2, ...
    FROM table_name
    ORDER BY column1, column2, ... ASC|DESC;

EXAMPLE

Consider a table 'EMPLOYEE' with following values:

EMP_ID NAME SALARY DEPT
1 MONA 200000 HR
2 SONA 100000 SALES
3 NAINA 50000 SALES
4 NAMAN 150000 PRODUCTION


TO Sort the table from highest to lowest salary: 
    SELECT * FROM EMPLOYEE ORDER BY SALARY DESC ;
