SELECT COMMAND

The most commonly used SQL command is SELECT command. The SELECT statement is used to query or retrieve data from a table in the database. A query may retrieve information from specified columns or from all the columns in the table.


SYNTAX: 

    SELECT column_list from table_name;
    [WHERE CLAUSE];
    [GROUP BY CLAUSE];
    [HAVING CLAUSE];
    [ORDER BY CLAUSE];
    The code within brackets is optional.

EXAMPLE

Consider a table STUDENT. To list information about all employees from STUDENT table. 
select * from STUDENT;
