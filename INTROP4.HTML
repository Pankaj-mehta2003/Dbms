%TYPE

%TYPE is used to declare variables whose data type is the same as a specific column in a table or cursor variable. It is particularly useful when you want to ensure that the data type of a variable matches the data type of a column, reducing the risk of data type mismatch errors.

EXAMPLE 

    NAME EMP.ENAME%TYPE;
Here, the variable Name will be of the same size and type as that of the ENAME column of the EMP table.


%ROWTYPE

%ROWTYPE is used to declare record variables whose structure matches the structure of a row in a table or a result set of a query. It is particularly useful when you want to work with entire rows of data without specifying individual column names.

EXAMPLE 

    ROW EMP%ROWTYPE;
Here, thw variable ROW will be a composte variable, consisting of the column names of the table as its members. To refer to a specific variable(say SAL), the following syntax will be used:
ROW.SAL:=50000;
